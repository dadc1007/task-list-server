# ¿Qué es mi producto y para que sirve?

Nuestro producto es un servidor para gestión de tareas que permite crear, consultar, actualizar y eliminar pendientes de forma rápida y confiable. Está pensado para equipos, estudiantes o personas que necesitan mantener el orden de sus actividades diarias.

Su propósito principal es ayudar a organizar el trabajo y reducir olvidos. En lugar de depender de notas sueltas o mensajes, los usuarios pueden centralizar sus tareas en un solo lugar y ver su estado en cualquier momento.

# ¿Cuáles son las funcionalidades más importantes y porque los usuarios las usarían?

- **Creación de tareas:** permite registrar nuevas actividades de inmediato, evitando que ideas o compromisos se pierdan.
- **Consulta de tareas:** muestra la lista completa y también permite buscar tareas específicas, lo que ahorra tiempo cuando se necesita información puntual.
- **Actualización de estado:** facilita marcar tareas como completadas o pendientes, ayudando a visualizar el avance real del trabajo.
- **Eliminación de tareas:** mantiene la lista limpia y actualizada, eliminando tareas que ya no son necesarias.
- **Validación de datos y manejo de errores:** el sistema detecta solicitudes incompletas o inválidas y responde de forma clara, lo que mejora la confianza y evita inconsistencias.

Los usuarios usarían estas funcionalidades porque les permiten tener control, claridad y seguimiento del trabajo sin complicaciones.

# ¿Qué tecnologías usaste y por qué?

- **Node.js:** se utilizó porque es eficiente para construir servicios rápidos que atienden muchas solicitudes.
- **Express:** se eligió por su simplicidad para crear rutas y lógica del servidor de manera ordenada.
- **JSON:** se usa como formato de intercambio de datos porque es estándar, ligero y fácil de integrar con aplicaciones web o móviles.

Estas tecnologías permiten construir una solución estable, escalable y fácil de mantener, ideal para crecer junto con las necesidades del cliente.
